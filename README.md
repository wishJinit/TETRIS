# TETRIS

### 개발기간

2017.09 ~ 2017.10

 ### IDE

Visual Studio

### Language

C++



### 기능

* 게임시작 : 테트리스 구현 (단계별 난이도 조정 구현)
* 게임방법 : 테트리스 진행방법 설명
* 게임랭크 : 파일입출력을 이용한 테트리스 게임 랭킹 구현
